Welcome to Dashboard page

The SCM Dashboard is a user-friendly web application designed to manage essential contact information for a Supply Chain Management (SCM) system. This Webiste allows users to efficiently save, update, and delete contact numbers for easy access and communication within the system. The platform provides a simple interface where users can quickly input new contact numbers, update existing ones, or remove outdated entries from their contact list. With dynamic, real-time updates to the contact list, users can manage their contact information in an intuitive, streamlined way. The website is designed to enhance operational efficiency in managing key contacts in the supply chain process, ensuring that communication is always up-to-date and reliable.
